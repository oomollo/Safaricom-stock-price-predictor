About Stock Predictor

Stock Predictor is a powerful tool designed to analyze and visualize stock market data, with a focus on Safaricom (SCOM) stock. This project was developed as part of an academic endeavor to explore data analysis and visualization techniques in the context of financial markets.

Features

  • Historical stock data visualization
  • Moving average calculations (100-day and 200-day)
  • Statistical analysis of stock performance
  • Interactive graphs for better understanding of stock trends

Technology Stack

  • Python: The core programming language used for data analysis and backend development
  • Django: A high-level Python web framework that encourages rapid development and clean, pragmatic design
  • Pandas: A fast, powerful, flexible and easy to use open source data analysis and manipulation tool
  • Matplotlib: A comprehensive library for creating static, animated, and interactive visualizations in Python
  • HTML/CSS: For structuring and styling the web pages

Developer

This project was developed by Olgah Awuor as part of a student project. It showcases the application of data analysis and web development skills in creating a practical tool for stock market analysis.
